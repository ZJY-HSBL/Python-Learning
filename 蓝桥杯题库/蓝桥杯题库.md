### 1259-奇怪的捐赠

**问题描述**

地产大亨 Q 先生临终的遗愿是：拿出 100100 万元给 X 社区的居民抽奖，以稍慰藉心中愧疚。

麻烦的是，他有个很奇怪的要求：

1. 100100 万元必须被正好分成若干份（不能剩余）。每份必须是 77 的若干次方元。比如：11 元，77 元，4949 元，343343 元，...
2. 相同金额的份数不能超过 55 份。
3. 在满足上述要求的情况下，分成的份数越多越好！

请你帮忙计算一下，最多可以分为多少份？

**运行限制**

- 最大运行时间：1s

- 最大运行内存: 128M

~~~python
import os
import sys

n = 1000000
num=[7**i for i in range(10)]

for a in range(1,5):
    for b in range(1,5):
        for c in range(1,5):
            for d in range(1,5):
                for g in range(1,5):
                    for e in range(1,5):
                        for f in range(1,5):
                            for h in range(1,5):
                                if a*num[0]+b*num[1]+c*num[2]+d*num[3]+g*num[4]+e*num[5]+f*num[6]+h*num[7]==n:
                                    print(a+b+c+d+g+e+f+h)
~~~
